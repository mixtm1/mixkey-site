Активация

Телефонная активация

Используйте эту инструкцию, когда Windows или Office предлагает активацию по телефону и показывает код установки.

Когда нужна активация по телефону

Телефонная активация может появиться, если онлайн-проверка Windows или Office не завершилась автоматически, ключ требует дополнительного подтверждения или продукт показывает окно с ID установки.

Важно

Не закрывайте окно активации, пока не получите ID подтверждения или не сделаете скриншот для поддержки MixKey.

Порядок действий

1

Как открыть окно активации

В Windows откройте раздел Активация. В Office откройте Word или Excel, затем Файл, Учетная запись и раздел активации.

Место для скриншота: окно активации
2

Как выбрать страну

В мастере телефонной активации выберите вашу страну или ближайший доступный регион из списка.

Место для скриншота: выбор страны
3

Как ввести ID установки

Сохраните все группы ID установки без изменений. Если обращаетесь в поддержку, приложите скриншот окна целиком.

Совет

Не переписывайте цифры вручную, если есть возможность отправить скриншот.

Место для скриншота: ID установки
4

Как получить ID подтверждения

Следуйте подсказкам мастера или обратитесь в поддержку MixKey и приложите скриншот с ID установки.

Место для скриншота: получение ID
5

Введите ID подтверждения

Введите полученные группы в поля мастера активации строго по порядку и нажмите Далее.

Место для скриншота: поля подтверждения
6

Что делать, если активация не проходит

Проверьте, что группы введены без ошибок. Если сообщение повторяется, сделайте скриншот и обратитесь в поддержку MixKey.

Место для скриншота: ошибка активации
