Переход Windows Home на Pro

Если у вас установлен выпуск Home, а ключ куплен для Pro, сначала смените редакцию Windows на Pro, затем активируйте систему купленным ключом.

Когда нужна эта инструкция

Она нужна, если в разделе О системе указано Windows 10 Home или Windows 11 Home, а в заказе указан ключ Windows Pro.

Важно

Технический ключ перехода не активирует Windows. Он только меняет редакцию Home на Pro. После перезагрузки обязательно введите купленный ключ Pro.

Порядок действий

1

Отключите интернет

Отключите Wi-Fi или сетевой кабель. Это помогает Windows сначала сменить редакцию, а не сразу проверять ключ онлайн.

2

Откройте активацию

В Windows 11 откройте Параметры, Система, Активация. В Windows 10 откройте Параметры, Обновление и безопасность, Активация.

3

Введите технический ключ

Нажмите Изменить ключ продукта и введите: VK7JG-NPHTM-C97JM-9MPGT-3V66T.

4

Перезагрузите компьютер

После смены редакции Windows попросит перезагрузку. Дождитесь завершения обновления и входа в систему.

5

Подключите интернет

После перезагрузки снова подключите интернет и откройте раздел Активация.

6

Введите купленный ключ Pro

Нажмите Изменить ключ продукта и введите ключ Windows Pro из заказа MixKey.

Если переход не сработал

Интернет включен

Windows сразу проверяет ключ онлайн

Отключите интернет и повторите ввод технического ключа.

Редакция не изменилась

Требуется повторная перезагрузка

Перезагрузите компьютер еще раз и проверьте выпуск Windows в разделе О системе.

Ошибка после перехода

Купленный ключ не введен

После смены редакции введите именно купленный ключ Windows Pro, а не технический ключ перехода.
