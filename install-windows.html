Windows

Установка Windows

Универсальная инструкция для Windows 10 Home, Windows 10 Pro, Windows 11 Home и Windows 11 Pro.

Подходит для

Windows 10

Home и Pro

Используйте эту инструкцию при чистой установке или переустановке Windows 10.

Windows 11

Home и Pro

Подходит для установки Windows 11 и подготовки системы к активации ключом.

Порядок установки

1

Скачайте Windows

Откройте раздел загрузок и скачайте официальный образ Windows или запасной дистрибутив, если официальный сайт недоступен.

Открыть загрузки
Место для скриншота: загрузка Windows
2

Создайте загрузочную флешку

Используйте официальный инструмент Microsoft или проверенную программу для записи образа на USB-флешку.

Важно

Все данные на флешке будут удалены. Сохраните нужные файлы заранее.

Место для скриншота: загрузочная флешка
3

Запустите установку

Перезагрузите компьютер, выберите загрузку с флешки и следуйте мастеру установки Windows.

Место для скриншота: мастер установки
4

Выберите редакцию Home или Pro

Выберите ту редакцию, для которой куплен ключ. Ключ Home не активирует Pro, а ключ Pro не предназначен для Home.

Место для скриншота: выбор редакции
5

Введите ключ продукта

Ключ можно ввести во время установки или после нее в разделе Параметры, Система, Активация.

Место для скриншота: ввод ключа
6

Проверьте активацию

После подключения к интернету откройте раздел Активация и убедитесь, что Windows активирована.

Место для скриншота: статус активации

Если возникла ошибка

Сверьте редакцию Windows с купленным ключом и проверьте подключение к интернету. Если ошибка остается, откройте раздел типовых ошибок или обратитесь в поддержку MixKey.
